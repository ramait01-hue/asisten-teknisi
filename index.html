Hal umum tentang cara kerja TeknisPro

Apakah aman memperbaiki elektronik sendiri?+
Untuk perbaikan ringan seperti mengganti sekring, membersihkan filter, atau mengecek kabel longgar — aman jika peralatan sudah dimatikan & dicabut. TeknisPro selalu mengingatkan batas aman setiap kali memberikan panduan.
Bisa upload foto kerusakan untuk diagnosa?+
Ya! Klik ikon 📎 di kotak chat atau langsung Ctrl+V / paste gambar dari clipboard. AI akan menganalisis gambar dan memberikan diagnosa lebih akurat berdasarkan kondisi visual yang terlihat.
Bagaimana cara set API Key Claude?+
Klik ikon 🔑 di pojok kanan atas, atau klik banner kuning yang muncul di chat. Masukkan API Key dari console.anthropic.com. Key disimpan di browser kamu, tidak dikirim ke server kami.
Apakah ada fitur input suara?+
Ya! Klik tombol 🎙️ di chat untuk berbicara langsung. Browser akan merekam suaramu dan mengubahnya menjadi teks otomatis. Membutuhkan izin mikrofon di Chrome atau browser modern.
Perlu login untuk menggunakan?+
Tidak wajib! Langsung chat tanpa login. Jika login, riwayat percakapan tersimpan ke cloud dan bisa dilihat, dilanjutkan, atau diekspor kapan saja dari perangkat manapun.
Apakah gratis sepenuhnya?+
Ya, TeknisPro sepenuhnya gratis. Kamu hanya perlu API Key Claude dari Anthropic (tersedia tier gratis). Tidak ada biaya berlangganan ke TeknisPro sendiri.
