Cara dapat API Key gratis

Panduan cepat untuk setiap provider AI

🔵 Google Gemini — Cara dapat API Key?+
1. Buka aistudio.google.com/app/apikey
2. Klik "Create API key" → pilih project
3. Salin key dan paste di TeknisPro via tombol 🔑

✅ Gratis 1.500 req/hari · Mendukung analisis foto/gambar · Tidak perlu kartu kredit
🔴 Groq — Cara dapat API Key? (Paling cepat!)+
1. Buka console.groq.com
2. Daftar gratis (bisa login Google)
3. Klik menu "API Keys" → "Create API Key"
4. Salin key (dimulai gsk_...) dan paste di TeknisPro

⚡ Gratis 14.400 req/hari · Super cepat · Model LLaMA 3.3 70B, Mixtral, Gemma
🟣 OpenRouter — Cara dapat API Key?+
1. Buka openrouter.ai → Login dengan Google
2. Klik "Keys" di menu → "Create Key"
3. Salin key (dimulai sk-or-...) dan paste di TeknisPro
4. Pilih model GRATIS: DeepSeek V3, LLaMA 3.3, Qwen3, Gemma 3

✅ 100+ model tersedia · Banyak yang gratis total
Provider mana yang paling direkomendasikan?+
🔴 Groq — Terbaik untuk percakapan sehari-hari, super cepat, limit besar
🔵 Gemini — Pilih ini jika perlu upload foto kerusakan untuk dianalisis
🟣 OpenRouter/DeepSeek — Untuk analisis mendalam dan teknis kompleks

💡 Tips: Simpan key untuk semua provider, lalu ganti-ganti sesuai kebutuhan!
Apakah API Key aman tersimpan di TeknisPro?+
Ya, 100% aman. API Key hanya tersimpan di browser kamu sendiri (localStorage) dan tidak pernah dikirim ke server TeknisPro. Kamu berhubungan langsung dengan server AI provider. Jangan pernah share key ke orang lain.
